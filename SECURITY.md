# Security Policy

## 1. Общая информация

Данный репозиторий является учебным проектом, выполняемым в рамках образовательной программы
ФЦТ МГУТУ им. К.Г. Разумовского.

Проект не предназначен для обработки персональных данных и не содержит конфиденциальной информации.
Используемые данные носят исключительно учебный и исследовательский характер.

## 2. Сообщения об уязвимостях и правила взаимодействия

Если в ходе работы с репозиторием были обнаружены уязвимости или ошибки,
рекомендуется создавать issue в данном репозитории.

Запрещается:
- публиковать персональные данные третьих лиц;
- размещать конфиденциальную информацию;
- использовать репозиторий для противоправных действий.

Все предложения и замечания рассматриваются в рамках учебного процесса.
