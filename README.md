# GeorgiiZhukov-mgutu-fct

Учебный open-source проект, выполняемый в рамках практической работы по дисциплине  
«Управление проектами в сфере информационных технологий».

Проект развивается на основе анализа выпускных квалификационных работ и научных публикаций,
посвящённых методам машинного обучения, интеллектуальному анализу данных и прогнозной аналитике.

---

## Научная основа

В рамках проекта был проведён анализ выпускных квалификационных работ и научных публикаций
(2023–2025 гг.), посвящённых применению интеллектуальных методов анализа данных.

Ключевые источники:
- ВКР «Применение технологий машинного обучения для анализа рынка недвижимости», 2025 г.
- ВКР «Разработка аппаратно-программного комплекса для анализа физической подготовки спортсменов
  с применением нейросетевых технологий», 2025 г.
- ВКР «Формализованное информационное описание для медицинской экспертной системы», 2023 г.

Ключевые методы:
- машинное обучение;
- анализ и обработка данных;
- интеллектуальные информационные системы;
- прогнозная аналитика.

Результаты анализа легли в основу формирования архитектуры и логики дальнейшего развития проекта.

---

## Как помочь?

Проект открыт для учебного и исследовательского участия.

### Как начать:
1. Сделайте fork репозитория.
2. Клонируйте репозиторий на локальный компьютер.
3. Установите зависимости (Python 3.10+).
4. Ознакомьтесь с кодом в директории `src/`.
5. Запустите тесты с помощью `pytest`.
6. Создайте pull request с улучшениями или предложениями.

Актуальные задачи размещаются во вкладке **Issues**.
